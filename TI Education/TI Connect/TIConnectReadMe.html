Known Issues

Installation
  • If you need to install an earlier version of the TI Connect™ software over TI Connect™ 1.6 software, you must first manually uninstall TI Connect™ 1.6. See the Help section entitled “Uninstalling the TI Connect™ software.”
  • If you encounter any operating problems with the TI Connect™ software after you install both the TI Connect™ software and the TI InterActive!™ software, re-install the TI Connect™ software.
TI DeviceExplorer
  • If the file name of an application variable (AppVar) or third-party data type you are transferring to a device begins with a digit, and that file exists on the connected device, TI DeviceExplorer does not ask if you want to replace it, it simply replaces it.
TI ScreenCapture
  • When you capture device screens, an error may occur if the handheld device does not have enough resources to respond to the request. For example, if the device user is performing operations or functions on the device when you try to capture the device screen.
Other TI Desktop Software
  • TI CellSheet Converter 1.0, TI LearningCheck Creator 1.0, TI NoteFolio Creator 1.0, TI PackageExplorer 1.0, TI StudyCards Creator 2.0, and TImeSpan Creator 1.0 may not be fully compatible with TI Connect™ 1.6. Each of these software packages has been updated to work with the TI-84 Plus Family and the TI-89 Titanium devices. These updates are, or will soon be, available on the web site listed below.
  • TI InterActive!™ 1.2 requires the installation of TI Connect™ 1.3.0.1 for communicating with connected devices. After installing TI Connect™ 1.6, TI InterActive!™ 1.2 will no longer allow connectivity with connected devices. TI InterActive!™ 1.3 is now available and is fully compatible with TI Connect™ 1.6. This update is available on the web site listed below.
  • TI Connect™ 1.6 will not install if you have TI-Navigator™ 1.0 installed. You must uninstall TI-Navigator 1.0 before installing TI Connect™ 1.6. If you have not upgraded to TI-Navigator 1.1, you can continue to use TI Connect™ 1.3.
Cables
  • If you plug in the USB computer cable or USB Silver Edition cable before installing the TI Connect™ software, Windows® may assign an incorrect driver for the cable. To correct the problem, see the Help section entitled “Re-installing your USB driver.”
  • If you are using the USB computer cable or USB Silver Edition cable, you may need to add the hardware through your system configuration menu. For plug-and-play operating systems, plug in the USB computer cable or USB Silver Edition cable after installing the TI Connect™ software and the system will find the appropriate driver automatically. For non-plug-and-play operating systems, you will need configure your system. Click Settings > Control Panel > Add Hardware.
  • Windows® 2000 users: While using a Black link or Gray link cable, the connected TI device may not respond normally to user input and the Automatic Power Down™ feature may not function correctly, causing excessive battery drain. This is directly related to the behavior of the port hardware and software on certain computers. This problem does not exist while using the USB computer cable or USB Silver Edition cable.
  • Windows® 2000 users: File transfers may fail when you use the Black link cable to connect the TI-92 to the computer. This is not a problem with the TI-92 Plus, the Voyage™ 200, or the TI-92 with the upgraded RAM module.
  • On some computers, when a TI device is connected with a Black link cable, the cursor on the device will not blink and the device will not automatically turn itself off during periods of inactivity. The device is, however, still usable with the TI Connect™ software.
TI-89 Family, TI-92 Family, and Voyage™ 200 Graphing Calculator
  • If some TI-89, TI-92, TI-92 Plus, or Voyage™ 200 fonts, such as the Greek alpha character, display incorrectly in the TI Connect™ software (sometimes they look like an empty box), you must re-install the TI-92 fonts on your computer.
  • When you transfer files to the TI-92, TI-92 Plus, and Voyage™ 200, it is possible to overwrite a file that is currently running on the device. Be sure that you close all Apps and files before transferring a file to a device.
  • The TI Connect™ software currently only supports Lab Reports as regular text files on the device with a file extension of .xxT.
  • On some computers, when a TI device is connected with a Black link cable, the cursor on the device will not blink and the device will not automatically turn itself off during periods of inactivity. The device is, however, still usable with the TI Connect™ software.
  • Windows® 2000 users: File transfers may fail when you use the Black link cable to connect the TI-92 to the computer. This is not a problem with the TI-92 Plus, the Voyage™ 200, or the TI-92 with the upgraded RAM module.
  • The TI Connect™ software does not let you transfer Flash applications to a TI-89 or TI-92 Plus with an operating system version of 2.04 or earlier. This is due to an issue that, under very rare circumstances, can cause the device to become unusable.
  • With some device operating systems, the TI Connect™ software is unable to determine the availability of newer versions of installed Flash applications on the TI Online Store. The affected operating systems are: TI-89: 2.05 and earlier TI-92 Plus: 2.05 and earlier. If you update your device operating system to the newest version, the Updates feature of the TI Connect™ software will then be able to notify you of available upgrades to your purchased Flash applications.
TI-73, TI-83 Family, and TI-84 Family
  • On some computers, when a TI device is connected with a Black link cable, the cursor on the device will not blink and the device will not automatically turn itself off during periods of inactivity. The device is, however, still usable with the TI Connect™ software.
  • The TI Connect™ software does not let you transfer Flash applications to a TI-83 Plus or TI-83 Plus Silver Edition with an operating system version of 1.13 or earlier. This is due to an issue that, under very rare circumstances, can cause the device to become unusable.
  • With some device operating systems, the TI Connect™ software is unable to determine the availability of newer versions of installed Flash applications on the TI Online Store. The affected operating systems are: TI-73: 1.5 and earlier TI-83 Plus: 1.13 and earlier TI-83 Plus Silver Edition: 1.13 and earlier. If you update your device operating system to the newest version, the Updates feature of the TI Connect™ software will then be able to notify you of available upgrades to your purchased Flash applications.
Firewalls
  • Firewalls may cause interference with the TI Connect™ software. They may make the TI Connect™ software unable to recognize attached devices and, in some instances, they may cause the TI Connect™ software to stop functioning entirely.
  • Some software products that provide firewall protection against Internet intrusion may display a warning that the TI Connect™ software is attempting to connect to the Internet, track your Internet activities, or report back to TI Connect™. This may be caused when the TI Connect™ software attempts to automatically check the TI Online Store for updates to the device operating system or Flash applications. The TI Connect™ software does not use cookies or any other means to track your computer or Internet activities.

System Requirements

To install and run the TI Connect™ software, you need the following:

  • Computer operating system: Microsoft® Windows® 98, ME, 2000, or XP.
  • Hard disk space: 50 MB hard disk space available (32 MB required for installation).
  • Web browser: Microsoft Internet Explorer 5.5 or greater (for functions that use your existing Internet connection).
  • TI Connectivity Cable: USB computer cable, USB Silver Edition cable, black link cable, or gray link cable.
  • TI handheld device: TI-73, TI-83, TI-83 Plus, TI-83 Plus Silver Edition, TI-84 Plus, TI-84 Plus Silver Edition, TI-86, TI-89, TI-89 Titanium, TI-92, TI-92 Plus, or a Voyage™ 200 Graphing Calculator. These devices must have the latest operating system installed. To update your device operating system, use Updates in TI Connect™ Home or TI OS Downloader in TI DeviceExplorer.
  • Access: Users with Windows® 2000 or XP need administrator privileges to install the TI Connect™ software.
  • Display properties: A color setting of at least 16-bit. The graphics in the TI Connect™ software have been optimized for use with a color-depth setting of at least 16-bit. You can adjust this in the Settings tab of your Display Properties window.

Note: If you have not installed the Microsoft .NET Framework update, you cannot install the Content tools. You can install Microsoft .NET by downloading it from Microsoft:

  1. When your computer is connected to the Internet, click Start > Windows Update.

    2. Your Web browser opens at the Windows Update Web site.

  2. Scan for available updates.
  3. Select Microsoft .NET Framework.
  4. Download and install the .NET Framework update.

Notes

  • If you are using Windows 2000 or Windows XP, you need administrator privileges to install the TI Connect™ software.
  • TI handheld devices must have the latest operating system (OS) installed. The TI Connect™ 1.6 install includes OS version 1.60 for the TI-73, OS version 1.16 for the TI-83 Plus and TI-83 Plus Silver Edition, OS version 2.21 for the TI-84 Plus and TI-84 Plus Silver Edition, OS version 3.00 for the TI-89 Titanium, and OS version 2.09 for the TI-89, TI-92 Plus, and Voyage™ 200. To update your device operating system, use Updates in TI Connect™ Home or TI OS Downloader in TI DeviceExplorer.
